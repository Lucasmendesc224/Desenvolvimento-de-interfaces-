Exemplo sobre web semântica

Este bloco refere-se a tag semantica <header>, que representa o cabeçalho de uma página, nele geralmente utilizaremos as tags de seção/titulo.

<header>

<Nav>

Neste bloco representamos a seção de navegação que utiliza a tag <nav>. Neste bloco definimos por exemplo os menus de navegação do nosso documento html. por exemplo:

  • inicio
  • cadastro
  • Relatórios
  • Movimentação
  • Sobre

<main>

A tag <main>, utilizada neste representa o conteudo principal da pagina. Nele teremos as tags semanticas<section>, <article>, <aside>.

<section>

Neste bloco estamos usando a tag semantica <section> para definir uma seção genérica de um documentos

HTML - web semantica

A web semantica refere-se ao uso do html e outras tecnologias para estruturar o conteudo de uma pagina de forma que ela seja compreendida por humanos quanto por maquinas. Isso significa usar elementos html que descrevam o significado do conteudo, em vez de apenas sua aparencia.

<article>

A tag semantica <article> define um conteudo independente, como uma postagem de bloco ou noticia.

<aside>

A tag semantica <aside> representa o conteudo auxiliar relacionado, mas não essencial, ele muitas vezes é utilizado como um menu auxiliar.

  • menu auxiliar 1
  • menu auxiliar 2
  • menu auxiliar 3
  • menu auxiliar 4

<footer>

A tag semantica footer é aquela que define um rodapé do documento html. Nela geralmente colocamos informações de fechamento do nosso documento, tais como nome, autor, data, etc.
