Exemplo 7 - html - listas ordenadas <ol> e listas não ordenadas < ul>

Este exemplo mostra como criar listas ordenadas e não ordenadas, incluindo subitens(listas aninhadas)

Lista ordenada <ol>

As listas ordenadas são aquelas que seguem uma ordem numericas ou por letras, são utilizadas para mostrar conteudo html em formato de lista.

Exemplo de lista ordenada <ol> juntamente com os elementos dessa lista que é representada pela tag <li>

  1. item 1
  2. item 2
  3. item 3
  4. item 4
  5. item 5

Exemplo 2 de listas ordenadas

  1. item 1
    1. subitem 1.1
    2. subitem 1.2
    3. subitem 1.3
    4. subitem 1.4
    5. subitem 1.5
  2. item 2
  3. item 3
