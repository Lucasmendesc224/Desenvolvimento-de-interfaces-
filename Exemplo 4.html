Exemplo 4 HTML - Estrutura Geral de um Documento HTML

Este exemplo mostra a estrutura mínima de um documento html. Aqui estão as príncipais tags utilizadas:

  • <!DOCTYPE html >:Define o tipo de documento html, ou seja a versão do html que será usada na pagina
  • < html>:a tag html é o elemento principal do documento html, ela define o container raiz de toda pagina
  • <head >:A tag head contém os metadados que serão interpretados pelo navegador. Nela encontraremos o titulo da página, links e arquivos para configuração do documento.
  • < body >: A tag body é responsavel pela exibição do conteudo visivel do documento, ou seja, tudo que estiver nesse bloco será visivel para o usuário.
